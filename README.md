# ThugFinance

Sistema de controle financeiro desenvolvido com React, Node.js, Express e SQLite.

## Funcionalidades

* Cadastro de usuários
* Login com JWT
* CRUD de categorias
* CRUD de transações
* Dashboard
* Controle financeiro

## Tecnologias

### Frontend

* React
* Vite
* CSS

### Backend

* Node.js
* Express
* SQLite
* JWT
* Bcrypt

## Deploy

Frontend:
https://thugfinance-app.vercel.app

Backend:
(Render)

## Como executar

### Backend

npm install

npm run dev

### Frontend

npm install

npm run dev
